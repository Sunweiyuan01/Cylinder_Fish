The Baseline_control folder contains the core implementation of the baseline CFD-coupled reinforcement learning framework. Among its main components, EnvFluent.py defines the simulation environment by connecting the reinforcement learning pipeline with the Fluent-based CFD solver. It is responsible for initializing the simulation, updating the environment state, applying locomotion-related control actions, calculating rewards, and managing episode transitions during training. The training.py file provides the main training pipeline for the baseline controller. It initializes the learning algorithm, creates the environment instance, configures training hyperparameters, and executes the policy optimization process. Together, these two files form the central interface between hydrodynamic simulation and reinforcement learning-based control.
